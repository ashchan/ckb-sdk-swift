# CKB SDK Swift

Swift SDK for Nervos [CKB](https://github.com/nervosnetwork/ckb).

## Prerequisites

To build CKB SDK, you'll need:

* Xcode 10.2 with Swift 5, and later
* CocoaPods or SPM
